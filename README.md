# AWS S3 Static Website (Free Tier)

Deployed a static website using **Amazon S3 static website hosting** as a cloud fundamentals project.

## What I built
- Public static website hosted on S3
- Configured S3 permissions using a bucket policy for public read access
- Set index and error documents for static hosting

## Tech
- AWS S3
- IAM / Bucket Policy
- HTML

## Live Site
- S3 Website Endpoint: <http://rithvik-cloud-static-site.s3-website-us-east-1.amazonaws.com/>

## Notes
This project is designed to stay within AWS Free Tier limits.
